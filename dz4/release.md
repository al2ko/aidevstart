**Что нового**
- В карточке номенклатуры добавлен признак «Большая этикетка». Для такой номенклатуры печатается увеличенная этикетка со штрихкодом крупного размера.
- В АРМ добавлено открытие отчета по остаткам при сканировании штрихкода склада или ячейки. Отчет открывается сразу с отбором по найденному складу или ячейке.
- В производственных АРМ добавлен отчет «Отказанные заявки». Вариант отчета настраивается в общих настройках.
- В настройках добавлена организация для документов. Создаваемые из АРМ документы, перемещения и резервирования теперь используют выбранную организацию.
- В регистре учета смен по препрегу добавлен учет по подразделениям. График смен можно заполнять с привязкой к подразделению, а упаковки получают смену с учетом участка.
- В АРМ рабочего по полиолу и в АРМ Мастера препрега добавлен ввод номера партии. При выпуске препрега проверяется актуальность номера партии, а при выпуске полиола - максимальное значение в партии. Упаковки создаются с указанной партией в АРМ.
- В ИГК добавлено хранение наименования склада БП.

**Исправления**
- Скорректирована приемка по заказу поставщику: количество сверх заказа выделяется отдельной строкой, чтобы не искажать закрытие заказа.
- Исправлено заполнение упаковок в резервировании запасов по документу перемещения. В резервирование попадают только упаковки, по которым выполнено перемещение.
- В АРМ кладовщика склада готовой продукции исправлено сопоставление остатков при проведении испытаний: проверка выполняется по паллете.
- Скорректирован расчет текущей смены и нумерации паллет с учетом подразделения и без учета помеченных на удаление упаковок.
- Исправлена привязка форм в производственных АРМ мастеров к своим рабочим местам.

**Изменения интерфейса**
- В формах отчетов производственных АРМ добавлена команда «Отказанные заявки».
- На начальных страницах АРМ обновлено отображение кнопок: команды растягиваются по ширине формы.
- В форме производства полиола добавлено поле «Номер партии».
- В настройках добавлены поля «Вариант отчета для АРМ Отказанные заявки» и «Организация для документов».
- В карточке номенклатуры добавлен флажок «Большая этикетка».
- В печатных формах этикеток упаковок уточнен вывод даты, времени и номера партии.

**Технические изменения**
- Обновлены роли и права доступа для АРМ, отчетов, упаковок, ИГК, испытаний, штрихкодов и связанных объектов.
- Служебные функции общего назначения перенесены в основную конфигурацию и используются из АРМ, регламентных заданий и обработок.
- Обновлен состав объектов конфигурации и расширения, включая перенос формы ввода даты и подключение модуля обмена через универсальный формат.
